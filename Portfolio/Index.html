My Project

Fake News Detection using ML

Created a machine learning system to identify false news based on the textual content of articles through the use of Python, Scikit-learn, and Pandas. Utilized natural language processing techniques like tokenization, removal of stopwords, and TF-IDF vectorization to prepare the dataset. Implemented and tested classification models which obtained more than 90% accuracy using the Logistic Regression model. Measured performance in terms of precision, recall, Fl-score, and confusion matrix, showing the capability of the model to be integrated into actual working misinformation detection systems.

Tech used: Python, Scikit-learn, and Pandas
